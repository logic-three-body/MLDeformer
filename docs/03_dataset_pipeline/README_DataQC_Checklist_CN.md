# 数据质量检查清单（Data QC）

## 1. 拓扑一致性
- [ ] 顶点数、索引数、拓扑顺序在全数据集中一致。
- [ ] UV/法线通道数量一致。
- [ ] Groom 的 group/curve/control point 索引一致。

## 2. 时间一致性
- [ ] 帧率统一（例如固定 30/60 FPS）。
- [ ] 无缺失帧、重复帧、时间戳倒退。
- [ ] 多源数据（动画/缓存/标签）帧号严格对齐。

## 3. 姿态覆盖（ROM）
- [ ] 常用姿态覆盖充分。
- [ ] 极限姿态有长尾覆盖。
- [ ] 各关键关节在训练集中的采样分布可视化已核对。

## 4. 异常姿态剔除
- [ ] 明显自穿插样本已剔除或降权。
- [ ] 非法反向关节折叠样本已处理。
- [ ] 极端抖动样本（速度/加速度异常）已处理。

## 5. Delta 与特征质量
- [ ] Delta 是基于同一基准姿态计算。
- [ ] 输入特征维度与 UE 模型输入配置一致。
- [ ] 归一化/裁剪策略在训练与推理保持一致。

## 6. UE 导入前检查
- [ ] Skeletal mesh 兼容检查通过。
- [ ] 训练帧数满足模型最小要求。
- [ ] Groom Binding 与目标网格匹配。

## 7. 训练后快速回归
- [ ] 训练姿态重建误差在预期范围。
- [ ] 未见姿态无明显爆炸伪影。
- [ ] 推理耗时与显存占用在预算内。
